**5S后自动关闭的广告**

```
<script>
        window.addEventListener('load',function(){
            var ad=document.querySelector('img');
            setTimeout(function(){
                ad.style.display='none';
            },9000);
            var btn=document.querySelector('button');
            btn.onclick=function(){
                ad.style.display='none';
                btn.style.display='none';
            }
        });
    </script>
    <button>点击</button>
    <img src="images//1.jpeg">
```
