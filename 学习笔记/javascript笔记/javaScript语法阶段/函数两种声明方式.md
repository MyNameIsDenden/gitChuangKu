2.函数表达式(匿名函数)

```javascript
var 变量名 =function(){};
var fun=function(){
	console.log('.....');
}
var fun=function(aru){
	console.log('.....');
    console.log(aru);
}
fun('pink');
//fun是变量名 不是函数名
//函数表达式里面存的是函数
```
